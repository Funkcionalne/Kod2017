Kaûdý máte debetnú Ëi kreditnú. Existuje jednoduchý algoritmus na overenie, Ëi ste sa pri zadávaní jej Ëísla (4 skupiny po 4 cifry, medzery ignorujte) nepomýlili:

int luhncheck(char *cn) {
        int i,x,dl,total=0,z;
        z=1; /* neparny pocet cislic, nasobime parne indexy */
        y=strlen(cn);
        if (y%2==0)
                z=0; /* parny pocet cislic, nasobime neparne indexy */
        for(i=z;i<y;i+=2) {
                x=((int)cn[i]-48)*2;
                if (x>9) x-=9;
                total+=x;
        }
        for (i=z^1;i<y;i+=2)
                total+=(int)cn[i]-48;
        return (total%10) ? 0:1;
}

 

Úmyselne uvádzam verziu v jazyku C, aby ste pochopili krásu funkcionálneho programovania. Algoritmus môûete z kódu pochopiù, alebo naštudovaù si ho inde.
